# LocalDrop ‚ö°

**LocalDrop** is a lightweight local network file sharing utility. It allows for quick file and text transfers between a PC and any device (Smartphone, Tablet, or other PC) connected to the same Wi-Fi network.

## üìå Overview

This tool turns your computer into a local server, enabling bidirectional transfers without cloud dependency. It features a modern dark-themed GUI and supports secure HTTPS connections for private browsing.

* **Language:** The application interface and documentation are in **French**.
* **Status:** This project is considered stable and **will likely not receive further updates**.

## ‚ú® Key Features

- **Bidirectional**: "Send" mode to host files or "Receive" mode to accept uploads.
- **Security**: Optional password protection (`bcrypt`) and SSL/TLS support.
- **Easy Access**: Instant pairing via QR Code and local URL.
- **No Installation Required (Client-side)**: Other devices only need a web browser.
